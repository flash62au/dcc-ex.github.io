All aboard! Take control of your model railroad with this elegantly designed app that brings vintage charm to modern model train control. Whether you're a seasoned engineer or just starting your journey in model railroading, ThrottleCard delivers powerful features wrapped in a beautiful, easy-to-use interface.


KEY FEATURES:


* Control multiple trains simultaneously with intuitive multi-throttle support

* Beautiful vintage-inspired gauges and controls with realistic animations

* Game Controller Support! Easily drive your Trains with a PlayStation, Xbox, or Nintendo Switch Controller with pre-mapped buttons

* Easily create consists on the fly right from within the app

* Automatic command station discovery

* Comprehensive locomotive function control

* Full turnout (switch) control

* Haptic Feedback for intuitive responsiveness and operation

* Mirror-Mode mode for comfortable operation in either hand

* Multiple classic railroad-inspired themes

* iPad support


With many more exciting features (including route support, native DCC-EX support, and a few surprises) in the roadmap!

To use ThrottleCard, simply connect to any WiThrottle Protocol enabled command station including JMRI, DCC-EX, and various commercial offerings.

ThrottleCard can be used for free without a subscription to control the speed, direction, and functions of a single locomotive at a time, throw turnouts, and change the theme.

Subscribe to Premium for $0.99/Month or $9.99/Year - 16% off - to unlock multiple throttles (cards), multiple locomotives in a consist, game controllers, and to support the future development of additional, exciting features! We'll even give you a week free to try it out.

We're committed to making the best tools possible for the model railroad community and would love to hear your feedback on the features you'd like to see next.

Get started with the latest in model railroading today!
