Take control of your model railroad with this elegantly designed app that brings vintage charm to modern model train control. Whether you're a seasoned engineer or just starting your journey in model railroading, |ThrottleCard| delivers powerful features wrapped in a beautiful, easy-to-use interface. You can connect ThrottleCard to |JMRI| and connect |JMRI| to |EX-CS|, or you if you are not going to use |JMRI|, you can connect directly to the Command Station if you install a WiFi board.

Please visit their website: https://throttlecard.com |EXTERNAL-LINK|

You can find it in the App Store: `ThrottleCard <https://apps.apple.com/app/apple-store/id6739982126?pt=451685&ct=dcc_ex&mt=8>`_ |EXTERNAL-LINK|

.. _throttlecard-features:

Features
=========

* Control multiple trains simultaneously with intuitive multi-throttle support
* Beautiful vintage-inspired gauges and controls with realistic animations
* Game Controller Support! Easily drive your Trains with a PlayStation, Xbox, or Nintendo Switch Controller with pre-mapped buttons
* Easily create consists on the fly right from within the app
* Automatic command station discovery
* Comprehensive locomotive function control
* Full turnout control
* Haptic Feedback for intuitive responsiveness and operation
* Mirror-Mode mode for comfortable operation in either hand
* Multiple classic railroad-inspired themes
* iPad/macOS support

With many more exciting features (including route support, native DCC-EX support, and a few surprises) in the roadmap!

.. _throttlecard-requirements:

Requirements
=============

* A |EX-CS| (Mega based for WiFi)
* An iOS/macOS device including iPhone, iPad or M1 Mac
* A WiFi Shield (or other ESP8622 solution) if you want to connect using WiFi :doc:`Wifi Setup </ex-commandstation/diy/wifi-setup>`

.. _throttlecard-operation:

Operation
==========

To use WiFi, make sure you have a WiFi enabled |EX-CS| as described in the :doc:`Wifi Setup </ex-commandstation/diy/wifi-setup>` section.

* Open the network settings on your phone
* Change to same network of the PC that |JMRI| or your |WiThrottle Server| is on
* Start the |ThrottleCard| App
* |ThrottleCard| will search for the |WiThrottle Server| on the |EX-CS| and show in the connections list
* If your |WiThrottle Server| is not showing in the list, you can manually add it by entering the IP address of the |EX-CS| and the port number - usually 12090 or 2560

To use ThrottleCard, simply connect to any WiThrottle Protocol enabled command station including JMRI, DCC-EX, and various commercial offerings.
